# Sistema de Controle de Tarefas

Sistema para gerenciamento de tarefas com integração entre backend e banco de dados.

## Objetivo

Permitir o cadastro, visualização e controle de tarefas por meio de uma aplicação conectada a um banco de dados relacional.

## Funcionalidades

* Criar tarefas
* Listar tarefas
* Atualizar status das tarefas
* Deletar tarefas

## Tecnologias

* Python (Flask)
* MySQL

## Destaque técnico

* Desenvolvimento de API REST
* Integração entre aplicação e banco de dados
* Implementação de operações CRUD
* Estruturação de banco de dados relacional

## Como executar

1. Criar o banco de dados:

   * Executar o script SQL (`database.sql`) ou criar manualmente o banco `tarefas_db`
2. Instalar as dependências:

   * pip install flask mysql-connector-python
3. Executar o backend:

   * python app.py
4. Acessar no navegador:

   * http://127.0.0.1:5000/tarefas
